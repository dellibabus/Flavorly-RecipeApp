# Flavorly – Recipe App

A modern recipe mobile app built using **React Native (Expo)** with clean UI, dark/light theme support, and real-time recipe data.

---

## Features

* Browse recipes from API
* Search recipes
* Category filtering
* Favorites (saved locally)
* Dark / Light mode toggle

---

## Tech Stack

* React Native (Expo)
* NativeWind (Tailwind CSS)
* Zustand
* Axios
* React Navigation

---

## Setup

```bash
git clone https://github.com/dellibabus/Flavorly-RecipeApp.git
cd Flavorly-RecipeApp
npm install
npx expo start
```

---

## Environment

Create `.env` file:

```env
EXPO_PUBLIC_API_BASE_URL=https://www.themealdb.com/api/json/v1/1
```
